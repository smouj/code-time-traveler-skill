# Viajero Temporal del Código

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Code Time Traveler para OpenClaw.

## Qué Resuelve
Analiza el historial de git para identificar cuándo se introdujeron errores y sugiere puntos de corrección óptimos.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque de seguridad primero
- Soporte para reversión

## Uso
/code-time-traveler

## Ejemplo
```
Analizar código para detectar problemas de análisis.
```

## Instalación
Esta habilidad está automáticamente disponible en OpenClaw.
